# B站课程信息提取器

这是一个用于 Tampermonkey 的脚本，专门提取 B 站视频页面中的课程列表信息，适合用来生成学习计划、统计时长和快速复制课程内容。

## 功能

- 在 B 站视频页面上自动添加“📋 提取课程”按钮
- 支持提取课程标题和每一项课程的名称、时长
- 自动统计课程总数与总时长
- 可选是否包含课程地址链接
- 提供多种导出格式：
  - 简洁文本
  - 表格格式
  - Markdown
  - JSON
  - CSV
- 一键复制结果到剪贴板
- 当前页面结构不变时自动重新插入按钮

## 使用方式

1. 将脚本安装到 Tampermonkey 中。
2. 打开 `https://www.bilibili.com/video/*` 页面。
3. 点击页面中自动添加的“📋 提取课程”按钮。
4. 在弹出的导出面板中选择导出格式。
5. 可勾选“包含课程地址”后复制结果。

## 适用场景

- 快速复制 B 站课程目录用于笔记整理
- 导出课程清单到 Markdown、Excel、JSON 等格式
- 统计学习资料总时长
- 生成学习计划、课程摘要

## 备注

脚本依赖页面中课程项的结构为 `.video-pod__item`，并从 `.title-txt` 和 `.stat-item.duration` 中提取名称与时长。若页面样式更新，可能需要调整选择器。
