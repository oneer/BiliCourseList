# BiliCourseList

> 自己在 B 站学习课程时，总觉得视频播放器的课程记录和选集管理做得不够方便，想要整理一份清晰的课程计划表、安排学习进度时，手动复制课程名称和时长又特别麻烦。这个脚本就是为了解决这个问题——一键提取课程信息，方便做学习规划。

一个用于 Tampermonkey 的脚本，提取 B 站视频页面中的课程列表信息，适合用来生成学习计划、统计时长和快速复制课程内容。

## 功能

- 在 B 站视频页面上自动添加"📋 提取信息"按钮
- 支持提取课程标题和每一项课程的名称、时长
- 自动统计课程总数与总时长
- 提取当前页面地址（去除查询参数）
- 提供多种导出格式：
  - 简洁文本
  - 表格格式
  - Markdown
  - JSON
  - CSV
- 一键复制结果到剪贴板
- 页面结构变化时自动重新插入按钮

## 使用方式

1. 将脚本安装到 Tampermonkey 中。
2. 打开 `https://www.bilibili.com/video/*` 页面。
3. 点击页面中自动添加的"📋 提取信息"按钮。
4. 在弹出的导出面板中选择导出格式并复制。

## 适用场景

- 快速复制 B 站课程目录用于笔记整理
- 导出课程清单到 Markdown、Excel、JSON 等格式
- 统计学习资料总时长
- 生成学习计划、课程摘要

## 备注

脚本依赖页面中课程项的结构为 `.video-pod__item`，并从 `.title-txt` 和 `.stat-item.duration` 中提取名称与时长。若页面样式更新，可能需要调整选择器。
